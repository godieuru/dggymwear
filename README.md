
# DGGYMWEAR

Este proyecto es un ecommerce de ropa deportiva y accesorios. 


## Objective 

Mi obejetivo dentro del proyecto es que todo lo que se pueda hacer click funcione, asi como tambien se envie email tanto para cambiar el password como cuando se envia la wishlist, tambien cambia el saludo al hacer login.

## Run Locally

Clone the project

```bash
  git clone https://github.com/godieuru/dggymwear.git
```

Go to the project directory

```bash
  cd dggymwear      
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
